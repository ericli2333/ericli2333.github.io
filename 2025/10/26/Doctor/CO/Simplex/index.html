我们将分为几个部分来详细解释：

  1. 对偶理论简要回顾
  2. 检验数的对偶解释：“ Reduced Cost ”的真正含义
  3. 检验数的计算原理
  4. 一个完整的例子

1. 对偶理论简要回顾

任何一个线性规划问题（称为原始问题）都有一个与之相伴的另一个线性规划问题（称为对偶问题）。

  • 原始问题：在资源限制下最大化收益。

    • 决策变量：生产各种产品的数量。
    • 目标函数：总收益最大化。
    • 约束条件：消耗的资源不能超过可用量。

  • 对偶问题：在保证“价值”不低于“收益”的前提下，最小化总价值。

    • 决策变量：$y_i$，表示第 $i$ 种资源的影子价格。即每增加一单位该资源，所能带来的目标函数（收益）的增量。
    • 目标函数：总价值最小化。总价值 = 可用资源量 × 其影子价格。
    • 约束条件：生产一种产品所消耗资源的“总价值”（资源量×影子价格）必须至少等于该产品带来的单位收益。否则，如果生产该产品消耗的资源价值低于其收益，那么生产它就是“划算的”，这个方案就不应被认为是最优的资源估价方案。

关键定理（强对偶定理）：如果原始问题有最优解，那么对偶问题也有最优解，且两者的最优值相等。

这意味着在最优解时，资源的总价值等于产品的总收益。任何不是“等值估价”的方案都不是最优的。

2. 检验数的对偶解释：“Reduced Cost”的真正含义

现在我们进入核心。在单纯形表中，目标函数行（即检验数行）的系数有一个专门的名字：Reduced Cost。

它的完整定义是：Reduced Cost = 原始成本 - 资源隐含成本

让我们来拆解这个公式：

  1. 原始成本：在目标函数 $Z = c_1x_1 + c_2x_2 + … + c_nx_n$ 中，变量 $x_j$ 的系数 $c_j$。它表示生产一单位产品 $j$ 能带来的直接收益。

  2. 资源隐含成本：要生产一单位产品 $j$，需要消耗各种资源。假设生产一单位 $x_j$ 需要消耗 $a_{1j}$ 单位的资源1，$a_{2j}$ 单位的资源2，…，$a_{mj}$ 单位的资源 $m$。
    当前，我们对这些资源的估价（影子价格）分别为 $y_1, y_2, …, y_m$（这些 $y_i$ 正是对偶变量的值）。
    那么，生产一单位产品 $j$ 所消耗资源的总价值就是：
    $a_{1j}y_1 + a_{2j}y_2 + … + a_{mj}y_m$
    这部分可以理解为生产该产品的机会成本。

  3. Reduced Cost：因此，检验数 $\bar{c}j$ 的完整表达是：
    $$\bar{c}j = c_j - \sum{i=1}^m a    {ij}y_i = c_j - (a_{1j}y_1 + a_{2j}y_2 + … + a_{mj}y_m)$$
    即： Reduced Cost = 单位直接收益 - 单位机会成本

它的经济学解释是：生产一单位产品 $j$ 所带来的净收益。

  • 如果 $\bar{c}_j > 0$：意味着直接收益大于机会成本，生产它是有净收益的，应该生产（即让它进基）。
  • 如果 $\bar{c}_j = 0$：意味着直接收益正好等于机会成本，处于最优平衡状态。
  • 如果 $\bar{c}_j < 0$：意味着直接收益小于机会成本，生产它是净亏损的，不应该生产（即让它保持为非基变量）。

注意：在标准形式的单纯形表（目标函数表示为 $Z - c_1x_1 - c_2x_2 - … = 0$）中，这个逻辑是反的：

  • $\bar{c}j = (c_j - \sum a{ij}y_i) > 0$ 在表中显示为正数，但意味着净收益为正。
  • 然而，我们通常会把检验数行写成 $Z + (-c_1)x_1 + (-c_2)x_2 + … = 0$ 的形式。此时，我们看到的检验数实际上是 **$- \bar{c}_j$**。
  • 所以，在常见的单纯形表中，我们看到的是：
    • 负的检验数 $\iff$ $\bar{c}_j > 0$ $\iff$ 净收益为正
    • 正的检验数 $\iff$ $\bar{c}_j < 0$ $\iff$ 净收益为负

这就是为什么在最大化问题中，我们要选择检验数为负的变量进基，因为它的净收益为正。

3. 检验数的计算原理

在单纯形法的迭代过程中，我们是如何实际计算出这些检验数的呢？

原理：对偶变量 $y_i$ 的值，就是单纯形表中目标函数行对应于初始基变量所在列的系数的相反数。

假设我们的初始模型是：
Max $Z = c^Tx$
s.t. $Ax \leq b$
$x \geq 0$

我们添加松弛变量 $s$，得到 $Ax + Is = b$。初始的基变量就是这些松弛变量 $s$。

构建初始单纯形表：

$x$ $s$ RHS
Z $-c$ 0 0
$s$ A I $b$

此时，初始基变量是 $s$。根据对偶理论，这些松弛变量对应的**对偶变量就是影子价格 $y_i$**。而在初始表中，目标函数行下，基变量 $s$ 所在的列，其系数是 0。

当我们进行枢轴变换，更换基变量后，基变成了 $x_B$（一部分是原来的决策变量，一部分可能是松弛变量）。一个重要结论是：在当前表中，目标函数行下，初始单位矩阵所在列（即初始松弛变量对应的列）的系数，取相反数，就是当前解对应的对偶变量的值。

即，如果当前表中，初始松弛变量 $s_1, s_2, …, s_m$ 所在的列的检验数分别为 $\bar{c}{S1}, \bar{c}{S2}, …, \bar{c}{Sm}$，则有：
$$(y_1, y_2, …, y_m) = (-\bar{c}{S1}, -\bar{c}{S2}, …, -\bar{c}{Sm})$$

那么，对于任何一个非基变量 $x_j$，它的检验数 $\bar{c}_j$ 的计算公式就是：
$$\bar{c}j = c_j - \sum{i=1}^m a_{ij}y_i$$
这完全符合我们上面推导的公式。在单纯形法的行变换中，执行枢轴运算（高斯消元）的目的，从对偶的角度看，就是在不断地更新对偶变量 $y_i$ 的值，并重新计算所有变量的 Reduced Cost，直到所有 $\bar{c}_j \leq 0$（在最大化问题中，表现为表中检验数 $\geq 0$），即所有产品的净收益非正，此时达到最优。

4. 一个完整的例子

考虑一个非常简单的例子：
原始问题：
Max $Z = 3x_1 + 2x_2$
s.t.
$2x_1 + x_2 \leq 18$
$2x_1 + 3x_2 \leq 42$
$3x_1 + x_2 \leq 24$
$x_1, x_2 \geq 0$

对偶问题：
Min $W = 18y_1 + 42y_2 + 24y_3$
s.t.
$2y_1 + 2y_2 + 3y_3 \geq 3$
$y_1 + 3y_2 + y_3 \geq 2$
$y_1, y_2, y_3 \geq 0$

初始单纯形表（添加松弛变量 $s_1, s_2, s_3$）：

$x_1$ $x_2$ $s_1$ $s_2$ $s_3$ RHS
Z -3 -2 0 0 0 0
$s_1$ 2 1 1 0 0 18
$s_2$ 2 3 0 1 0 42
$s_3$ 3 1 0 0 1 24
  • 初始对偶解：基变量是 $(s_1, s_2, s_3)$，它们在Z行的系数是 $(0, 0, 0)$。因此，当前的对偶变量取值为 $(y_1, y_2, y_3) = (-0, -0, -0) = (0, 0, 0)$。这意味着资源目前被认为是“免费的”，没有价值。
  • 计算 $x_1$ 的检验数：$\bar{c}1 = c_1 - (a{11}y_1 + a_{21}y_2 + a_{31}y_3) = 3 - (20 + 20 + 3*0) = 3$。净收益为3。
    在表中，它显示为 -3（因为表中是 $- \bar{c}_j$）。这个巨大的负检验数（对应巨大的正净收益）告诉我们 $x_1$ 应该进基。

经过几次迭代达到最优后，假设最终单纯形表为：

$x_1$ $x_2$ $s_1$ $s_2$ $s_3$ RHS
Z 0 0 1/2 0 1 36
$x_2$ 0 1 3/2 0 -1 6
$s_2$ 0 0 -4 1 2 24
$x_1$ 1 0 -1/2 0 1 6

  • 读取对偶解：最终表中，初始基变量 $s_1, s_2, s_3$ 对应的列是第3、4、5列。它们在Z行的系数分别是 $1/2$, $0$, $1$。
    因此，最优对偶变量为：
    $y_1 = -(\bar{c}{s1}) = - (1/2) = 1/2$
    $y_2 = -(\bar{c}    {s2}) = - (0) = 0$
    $y_3 = -(\bar{c}_{s3}) = - (1) = 1$
    影子价格 $(y_1, y_2, y_3) = (0.5, 0, 1)$。

  • 验证检验数（Reduced Cost）：

    • 对于 $x_1$ 和 $x_2$（基变量），其检验数必然为0（表中已显示为0）。
    • 对于 $s_1$（非基变量），我们可以用对偶变量验证其检验数：
      $\bar{c}{s1} = c{s1} - (a_{1,s1}y_1 + a_{2,s1}y_2 + a_{3,s1}y_3)$
      松弛变量 $s_1$ 在原始目标函数中的系数 $c_{s1} = 0$。
      它在三个约束中的系数向量是 $[1, 0, 0]^T$（因为它是第一个约束的松弛变量）。
      所以 $\bar{c}_{s1} = 0 - (1y_1 + 0y_2 + 0y_3) = 0 - (10.5) = -0.5$。
      而表中 $s_1$ 下的检验数正好是 0.5（即 $-(-0.5) = 0.5$）。验证成功。

这个计算过程清晰地展示了检验数是如何通过对偶变量（影子价格）计算出来的，其本质就是衡量一项活动（生产一个产品）的直接收益与消耗资源的机会成本之间的差额。单纯形法就是在寻找一个既满足原始可行性（资源约束），又满足对偶可行性（净收益非正，即 $\bar{c}_j \leq 0$）的完美解。
